README
----------


We have completed deployment of the best ResNeXt model in the local machine, the experiments results are
present in the xlsx file uploaded.

The file resnext.py consists of the flask deployment blueprint with the option of inputting the image url
and getting the prediction through the model deployed.

The file app_updated.py consists of the flask deployment blueprint with the added feature of browsing for 
an image from the local  machine and uploading and then getting the prediction through the model deployed. 


The tested jpeg images are also uploaded for reference.
